# 记忆插件 (Memory)

## 简介
允许 AI 在对话中保存、读取和删除文本记忆。

## 调用格式
AI 在回复中插入以下标签即可调用：

保存记忆:
```plaintext
<mem:键名>要保存的内容</mem:键名>
```

删除记忆:
```plaintext
<mem-del:键名>
```

## 示例

用户: 记住我喜欢暗色主题
AI: 好的，已记住你的偏好。<mem:主题偏好>喜欢暗色主题</mem:主题偏好>

用户: 忘记我的主题偏好
AI: 已删除该记忆。<mem-del:主题偏好>

## 执行流程
1. AI 在回复中输出记忆标签
2. 系统自动识别并调用记忆API保存/删除
3. 标签不会显示给用户
4. 系统显示保存/删除确认消息

## 启用/禁用
在输入框的深度思考弹窗 → 工具链 → 记忆 中切换。
