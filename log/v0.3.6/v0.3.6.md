# v0.3.6

## 功能新增

1. **Vue + TypeScript 组件系统**
   - 新增 `static/vue/app.ts` 作为 Vue 3 组件入口
   - 文件浏览器组件：Windows 风格的缩略图网格排列、右键菜单（查看/删除/重命名/下载）
   - 历史对话列表组件：自动排序，新消息自动置顶
   - 通过 esbuild 编译为 `static/vue/app.js`，不依赖现有 JS 文件
   - 构建命令: `npm run build-vue`

2. **文件浏览器增强**
   - 大缩略图排列（grid 布局，自动填充）
   - 右键上下文菜单：查看文件、重命名、下载、删除
   - 重命名弹窗对话框
   - 暗色模式支持

3. **历史对话排序优化**
   - Vue 组件每 2 秒轮询检测新消息
   - 有新的用户/AI 输出后自动将该对话置顶

4. **模型输出保底机制**
   - 检测 AI 是否仅在深度思考字段输出内容而正式输出为空
   - 自动重新调用模型一次，覆盖上次深度思考，补齐正式输出

## 功能优化

1. **提示词系统重构**
   - 命令执行 → `config/Plugin/Command.md`
   - 记忆 → `config/Plugin/Memory.md`
   - 模型提问 → `config/Plugin/Ask.md`
   - Agent → `config/Plugin/Agent.md`
   - 思维链注入 → `config/guidelines/cothink.md`
   - `getPluginPrompts()` 同时读取 `Plugin/` 和 `guidelines/` 目录

2. **实时插件状态 system 消息**
   - 每次请求前插入一条 system 消息显示当前所有插件开关状态
   - 随开关变化自动更新

3. **代码块深色主题**
   - 代码块背景 `#2d2d2d`，工具栏 `#363636`
   - Token 语法高亮色（注释/数字/关键字/字符串）
   - 插件输出块（plugin-block）匹配深色风格

4. **深度思考弹窗精简**
   - 移除了"思维链注入"和"压缩旧执行"开关
   - 两个开关均移至设置面板保留

## 问题修复

1. 修复 ask 块 `<q>` 标签格式兼容（同时支持 `<q=内容>` 和 `<q>内容</q>`）
2. 修复 `<mem:键名>内容</mem>` 闭合标签解析
3. 移除 UAC 提权功能及相关代码
4. 修复 `.plugin-block-header` 悬停变色的样式问题
5. 移除 `c1=` 属性解析，回归纯文本选项

## 技术变更

- 新增 `package.json` scripts: `build-vue`
- 新增 `static/vue/` 目录及 Vue 3 组件
- `static/debug.js` 作为独立调试工具文件
