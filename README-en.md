## Implemented Plugin Types

### 1. Memory (fully implemented in v0.2.0)
The model can autonomously choose which memories to write. Memories will take effect across all conversations, provided you have enabled the memory toolchain rather than disabling it.

### 2. Command Execution (fully implemented in v0.2.0)
The model can autonomously execute Cmd commands and PowerShell commands. Also compatible on Linux platforms.

## Installation

Currently, `Fold.ai` is not available on any package manager platforms. The release version has been built as native JavaScript and can be run directly with Node.js:

```
#1. Manually build project dependencies
cd fold
npm install
Node bin/server.js

#2. Windows users can use `dependency_install.bat` for one-click startup
Double-click the .bat file

#3. Linux / MacOS users can use the .sh script to install dependencies and start
Double-click the .sh file
```
