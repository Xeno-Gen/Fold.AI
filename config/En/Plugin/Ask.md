[ask Question Block]
Plugin:3

- Ask the user: <ask><q=question><o1=option1><o2=option2><o3=option3></ask>

- <ask> supports 1 to 9 options, e.g. <o4=option4><o5=option5>. You can freely ask one or multiple questions depending on the situation.
- When you are in a scenario where you can ask the user questions, or have completed a task, etc., you can use the <ask> tag block to ask the user questions.
