**轻量级AI框架 · 纯净对话 · 插件可扩展**

## ✨ 特性

- 🤖 **多模型支持** - DeepSeek、Kimi、智谱、通义千问、ChatGPT、Gemini 等
- 🔌 **插件系统** - Python 函数插件，支持自定义扩展
- 🧹 **无提示词污染** - 对话纯净，无亢余提示词干扰
- 📁 **文件管理** - 公共文件夹读写、上传下载、目录管理
- 💬 **对话管理** - 历史会话、分支对话、消息编辑重新生成
- 👥 **用户系统** - 注册登录、私信聊天、在线状态
- 🛡️ **管理员控制** - 用户管理、插件管控、权限设置
- 🎨 **界面美观** - 深色模式、自定义主题、流畅动画

## 🚀 快速开始（Windows）

### 环境要求

- Node.js v24.11.1
- Python 3.12.10

### 一键启动

双击 **`点我启动.bat`**，自动安装依赖并启动服务器

### 手动启动

```bash
cd 目录
npm install
npm start
```

访问 `http://localhost:17923`

## 📁 项目结构

```
Fold-AI/
├── 点我启动.bat       # Windows 一键启动脚本
├── server.js          # 主服务端
├── Mod/               # 插件目录
├── data/              # 用户数据
├── public/            # 公共文件
├── ken/               # 须知文件
└── com/               # 配置目录
```
